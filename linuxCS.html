Cheat Sheet – Linux

Voici une compilation de commandes utiles que j’utilise régulièrement dans mes travaux sur Linux.

Navigation dans le système

pwd
cd /chemin/du/dossier
ls -la

Manipulation de fichiers

touch fichier.txt
cp source.txt destination.txt
mv ancien.txt nouveau.txt
rm fichier.txt
cat fichier.txt
nano fichier.txt

Gestion des permissions

chmod 755 script.sh
chown utilisateur:fichier

Utilisateurs et groupes

whoami
id
adduser nom_utilisateur
passwd nom_utilisateur

Réseau

ip a
ping google.com
netstat -tulpn
ss -tuln
curl ifconfig.me

Processus

ps aux
top
kill -9 PID

Recherche

find / -name fichier.txt
grep "mot" fichier.txt

Compression

tar -czf archive.tar.gz dossier/
unzip fichier.zip

Autres utiles

history
alias ll='ls -la'
sudo !!
